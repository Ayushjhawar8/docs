# Contributing to tscircuit

## Core Repositories

| Repository | Description | Skills | Issues |
|------------|-------------|---------|---------|
| [tscircuit/tscircuit](https://github.com/tscircuit/tscircuit) | Main monorepo with core packages for circuit building, React components, visualization, and routing | TypeScript, React, Electronics | [![GitHub issues](https://img.shields.io/github/issues/tscircuit/tscircuit)](https://github.com/tscircuit/tscircuit/issues) |
| [tscircuit/core](https://github.com/tscircuit/core) | Core electronics primitives and circuit validation | TypeScript, Electronics | [![GitHub issues](https://img.shields.io/github/issues/tscircuit/core)](https://github.com/tscircuit/core/issues) |
| [tscircuit/jscad-fiber](https://github.com/tscircuit/jscad-fiber) | React components for 3D CAD modeling using JSCAD | TypeScript, React, CAD | [![GitHub issues](https://img.shields.io/github/issues/tscircuit/jscad-fiber)](https://github.com/tscircuit/jscad-fiber/issues) |
| [tscircuit/builder](https://github.com/tscircuit/builder) | 3D component modeling and PCB visualization | TypeScript, React, CAD | [![GitHub issues](https://img.shields.io/github/issues/tscircuit/builder)](https://github.com/tscircuit/builder/issues) |
| [tscircuit/schematic-symbols](https://github.com/tscircuit/schematic-symbols) | Standard electrical symbol library | SVG, Electronics | [![GitHub issues](https://img.shields.io/github/issues/tscircuit/schematic-symbols)](https://github.com/tscircuit/schematic-symbols/issues) |
| [tscircuit/docs](https://github.com/tscircuit/docs) | Documentation at docs.tscircuit.com | Technical Writing, MDX | [![GitHub issues](https://img.shields.io/github/issues/tscircuit/docs)](https://github.com/tscircuit/docs/issues) |
